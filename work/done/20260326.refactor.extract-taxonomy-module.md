# refactor: Extract taxonomy module and replace isInline with isInlineFlow

Implements proposed item 1 from DISPLAY_CLASSIFICATION__WORK — replace `isInline` with `isInlineFlow` as a pure display predicate and clean up the module structure.

## What changed

### 1. New module: `src/lib/taxonomy.ts`

All `is*` classification predicates extracted from `traversal.ts` into a dedicated module. Organized with primitives at the top and derived predicates at the bottom:

**Primitives:** `isAlreadyFocusable`, `isIsland`, `isIgnorable`, `isFocusable`, `isToken`, `isAnchor`, `isInlineFlow`, `isImplicitLine`

**Derived:** `isLine`, `isCursorBoundary`, `isTransparentBlock`, `isLineSibling`

`taxonomy.ts` imports only from `constants.js` — no dependency on traversal or token.

### 2. `isInline` removed, `isInlineFlow` introduced

`isInline` was tangled — it checked focusable, island, implicit-line class, float, AND display in one function. Replaced with `isInlineFlow`, a pure display predicate that only checks CSS `display: inline` or `display: inline flow` plus float exclusion. No focusable/island/class checks.

Five call sites updated to use appropriate primitive combinations instead of `isInline`:

- **`traversal.ts` → `lineSiblingDescend`**: `isFocusable(n) && !isIsland(n) && !isImplicitLine(n) && isInlineFlow(n)`
- **`token.ts` → `tokenizeLineRec`**: `isFocusable(child) && !isIsland(child) && isInlineFlow(child)`
- **`token.ts` → `buildImplicitLine`**: `isFocusable(sib) && !isIsland(sib) && !isImplicitLine(sib) && isInlineFlow(sib)`
- **`token.ts` → `tagImplicitLines`** (two sites): same combination as `buildImplicitLine`

### 3. `traversal.ts` slimmed down

Re-exports all `is*` from `taxonomy.js` for backward compatibility. Retains only traversal logic: sibling helpers, `getLine`, `isSameLine`, LINE_SIBLING traversal functions.

### 4. `token.ts` imports split

Imports `is*` predicates from `taxonomy.js` and traversal helpers from `traversal.js`.

### 5. Tests reorganized

- Created `__tests__/taxonomy.test.ts` with tests for `isToken`, `isInlineFlow`, `isLine`
- Removed those describe blocks from `__tests__/lineSibling.test.ts`
- `lineSibling.test.ts` updated to import `is*` from `taxonomy.js`
